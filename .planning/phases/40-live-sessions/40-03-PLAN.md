  <files>src/components/session/LiveTimer.tsx, src/components/session/PresenceIndicator.tsx</files>
  <action>
Create **src/components/session/LiveTimer.tsx**:
- Props: startedAt: number, durationMs: number
- Uses useEffect with setInterval(1000ms) to compute remaining = Math.max(0, startedAt + durationMs - Date.now())
- Displays MM:SS format
- When remaining < 60000: add text-red-500 and animate-pulse classes
- When remaining === 0: show "Time's up" text
- Clean, minimal component. Export named function LiveTimer.

Create **src/components/session/PresenceIndicator.tsx**:

- Props: sessionId, phaseId, activePromptIds (from live state)
- Subscribes to api.course.sessionQueries.getPresence (facilitator-only query)
- For each active prompt, query coursePromptResponses to determine who submitted
- Display three groups with counts and user names:
  - Submitted (green check icon) — users with submitted response
  - Typing (yellow pencil icon) — users with presence status='typing' and no submitted response
  - Not Started (gray circle icon) — enrolled participants with neither presence nor submission
- Use Badge components with appropriate colors. Expand/collapse to show individual names.

Both components: named exports, no default exports. Use shadcn/ui (Badge) and lucide icons.
</action>
