**Time-to-session indicator (MOD-03):**
1. Create a helper function `daysUntilSession(sessionDate: number, now: number = Date.now()): number` that computes calendar-day difference using `startOfDay` normalization (per CONTEXT.md decision -- timezone-hardened):
```typescript
function daysUntilSession(sessionDate: number, now: number = Date.now()): number {
  const sessionDay = new Date(sessionDate)
  sessionDay.setHours(0, 0, 0, 0)
  const today = new Date(now)
  today.setHours(0, 0, 0, 0)
  return Math.ceil((sessionDay.getTime() - today.getTime()) / 86400000)
}
```
2. In the SessionTimeline component (or wherever sessions are rendered with their linked modules), for each session that is in the future, compute:
